# Paradise Nursery Shopping Application

Paradise Nursery is an online plant shopping application built using React and Redux.

## Features

- Landing page with company information
- Product listing page with multiple categories of plants
- Shopping cart functionality
- Add to cart, remove item, and update quantity
- Dynamic cart total calculation
- Redux state management

## Technologies Used

- React
- Redux Toolkit
- JavaScript
- CSS

## Project Structure

- AboutUs.jsx
- App.jsx
- ProductList.jsx
- CartItem.jsx
- CartSlice.jsx
